# MFN & FBA Weekly Detractor App

Streamlit web app for weekly MFN & FBA detractor analysis.

## Features
- Weekly detractor logic (Sunday-based)
- MFN & FBA separation
- SellerFlex exclusion for FBA
- Account Manager mapping via Excel
- AM-based filtering & download

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
